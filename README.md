# UNA - Sistemas Distribuídos e Mobile

## Lista de Exercícios 06

Aplicação Console em C# que consome a API do Pokémon.

### Funcionalidades

- Solicita o nome de um Pokémon
- Consome a API pública
- Exibe no console:
  - Id
  - Nome
  - Altura
  - Peso

### API utilizada

https://pokeapi.co/api/v2/pokemon/{nome}

### Como executar

Execute no terminal:

dotnet run

Digite o nome de um Pokémon.
